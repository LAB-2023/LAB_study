## 🎮 GIT FLOW

   * 개인 이름 브랜치를 만들어서 각자 개발사항을 추가

## 🎮 GIT COMMIT MESSAGE CONVENTION
   * [ADD] , [REFACTOR] , [DELETE] , [HOTFIX]
   * 커밋 메세지는 모두가 보고 백트래킹이 되어야 하기 때문에 문장 자체는 한국어로 구성 
   * EX) [ADD] 유저 로그인 토근화 기능 추가
